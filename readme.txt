Embed the Staffing Engine Chatbot on your Wordpress site

== Description ==

*Note:* To use this plugin you must be an existing Staffing Engine customer

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/staffing-engine-chatbot' directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Access settings for the plugin by going to Settings > Staffing Engine
1. At a minimum, enter the embed code to connect the chatbot to your live chat instance. All other configuration
options are optional. Contact our support team for assistance, or to obtain your live chat embed code.
